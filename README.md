# gogowisma138.com

Repository ini berisi materi dan file untuk landing page **gogowisma138.com**, yang kami bangun sebagai “wisma digital” bertema **Wisma138**.

Fokus utamanya:

- Menyajikan info dan update seputar dunia game dan hiburan online dengan bahasa yang santai.
- Membuat halaman yang ringan, cepat diakses, dan mudah dipahami pengunjung.
- Menjaga tampilan tetap bersih, modern, dan enak dibaca di desktop maupun mobile.

Struktur yang direncanakan:

- `index.html` – landing page utama untuk pengunjung gogowisma138.com  
- `assets/` – folder untuk gambar, icon, dan file pendukung  
- `css/` – stylesheet untuk layout & tampilan  
- `js/` – script kecil kalau dibutuhkan (misalnya tracking, interaksi ringan)

Ke depan, repo ini akan jadi tempat utama untuk menyimpan dan mengembangkan versi terbaru dari landing page gogowisma138.com sebelum di-upload ke hosting.
